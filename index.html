API Reference

Core endpoints required for C4 Protocol compliance:

Core Endpoints

GET /api/v1.0/running

List running instances with detailed status information

Response includes:
  • Instance metadata
  • Resource allocation
  • Network configuration
  • Billing information

POST /api/v1.0/register

Create new instances with specified configurations

Supports:
  • Custom resource allocation
  • Network configuration
  • Storage attachment
  • Metadata management

DELETE /api/v1.0/unregister

Remove instances and cleanup resources

Features:
  • Graceful shutdown
  • Resource cleanup
  • Billing termination
